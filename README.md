# ArrayList – аналог std::vector на C++



Учебный проект: собственная реализация динамического массива, максимально приближенная к `std::vector` по интерфейсу и поведению.



## Возможности

- Хранение элементов любого типа (шаблонный класс)

- Автоматическое управление памятью через `::operator new`/`delete`

- Placement new для конструирования элементов и явный вызов деструкторов

- Move-семантика (конструктор и оператор присваивания с перемещением)

- Поддержка `std::initializer_list`

- Полный набор методов: `push_back`, `pop_back`, `insert`, `erase`, `resize`, `reserve`, `clear`, `shrink_to_fit`, `emplace_back`, `assign`

- Итераторы с произвольным доступом и обратные итераторы

- Операторы сравнения (`==`, `!=`, `<`, `>`, `<=`, `>=`)

- SFINAE-защита от неоднозначности между диапазонными конструкторами и конструкторами с числом элементов



## Сборка и использование

1. Склонируйте репозиторий или скачайте заголовочный файл `arraylist.hpp`.

2. Подключите его в свой проект:

```cpp
#include "ArrayList.hpp"

 ```
